# 命运 Skill

当用户要求“命运 Skill”“命运skill”“社交直觉弱”“遇不到对的人”“抓不住对的人”“如何识别对的人”“如何创造美好 moment”“上帝视角”“当断则断”“男性价值”“强者转化”“价值修炼”“翻页判断”“通用关系修炼法”“第三视角修炼法”时，使用本 skill。

先读取：

```text
references/destiny-skill-universal-cultivation.zh-CN.md
```

## 核心定位

命运 Skill 不是恋爱技巧，也不是操控术。

它训练的是：

```text
看见命运给了什么题。
警惕自己在哪些幻觉里误判。
站到别人系统里看自己。
调节靠近与远离的距离。
创造能被记住的美好 moment。
```

一句话：

```text
不是抓住对的人，而是修出能识别、靠近、承接和放过对的人的能力。
```

## 输出格式

当用命运 Skill 分析一个人或一段关系事件时，按这个格式输出：

```text
命运题：
这次关系/社交事件在训练我什么？

当前信号：
对方给了什么信号，我给了什么信号？

坏乡警惕：
我有没有进入幻想乡或自大乡？

他心推演：
对方可能如何解释我？

距离判断：
该靠近、留白、降压，还是停止？

光环识别：
双方真实光环在哪里？

叙事 moment：
如果可以行动，什么低压力瞬间最值得创造？

价值修炼：
我缺的是外显、生存、情绪，还是思想价值？

翻页判断：
这件事还能现实修复，还是只剩自我修炼？

今日修行：
一个最小现实动作。
```

## 红线

```text
不把命运 Skill 用成操控术。
不把光环叙事用成包装骗术。
不把他心通用成读心和绕边界。
不把距离通用成冷暴力。
不把上帝视角用成审判众生的傲慢。
不把男性价值用成索取回应的资格证。
不把遇不到对的人，解释成世界亏欠自己。
不把抓不住对的人，解释成自己必须更用力抓。
```

如果出现明确拒绝、拉黑、恐惧反应、要求别联系、现实边界切断，本 skill 只允许服务于停止、复盘、自我修炼和能力迁移，不允许服务于换路接近、制造偶遇、借命运叙事追击。
