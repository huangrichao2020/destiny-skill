## 两套框架的关系

```text
命运 Skill（四通）：从命运事件切入，训练社交直觉和关系能力——行动框架。
四乡修行法门：从人性结构切入，诊断修行卡点和力量流向——诊断框架。
```

两者合用：先用四乡看清格局，再用四通给出行动。

## 核心定位

命运 Skill 不是恋爱技巧，不是操控术。四乡修行法门不是心理学术语，不是人格分类。

它们训练的是：

```text
看见命运给了什么题。
警惕自己在哪些幻觉里误判。
站到别人系统里看自己。
调节靠近与远离的距离。
创造能被记住的美好 moment。
不被四乡所转，是定力。能转他人四乡，是法力。
```

一句话：

```text
不是抓住对的人，而是修出能识别、靠近、承接和放过对的人的能力。
```

## 底层引擎：住相

四乡不是技巧，是四种人最容易"住进去"的相。住相 = 把暂时的真相当成永恒的真理，住在里面不走了。

住相闭环：**住相 → 排异 → 抬杠 → 情绪 → 偏见 → 更深的住相**。

每一乡分**表层住相**（意识到的）和**深层住相**（不自知的驱动力）。修行只在表层是治标，修到深层才是治本。

## 思维层次

- **形式逻辑**（多数人）：单变量、线性、凭感觉下判断。住相就是形式逻辑的产物。
- **辩证逻辑**（高手）：多变量、多维度、能同时持有矛盾。辩证逻辑天然破相。

形式逻辑的定力 = "我忍住了"但还是痛。辩证逻辑的定力 = "我根本不需要忍"。

## 四乡速查

| 乡 | 核心问题 | 定力修什么 | 法力修什么 | 形式逻辑 | 辩证逻辑 |
|---|---|---|---|---|---|
| 自大乡 | 我怎么看自己 | 放下我执，真正看见别人 | 让别人在你这里被看见、被托住 | "我对，你错" | "我可以同时对和错" |
| 幻想乡 | 我怎么想未来 | 不逃避眼前，分清规划与幻想 | 让别人看见与你相关的未来感 | "以后会更好" | "不确定，但我依然行动" |
| 光环乡 | 别人怎么看我 | 不被评价牵动，看人看本质 | 给别人舞台，让他发光 | "我是X" | "我是流动的、多面的" |
| 叙事乡 | 我为什么在这里 | 不被付出绑架，主角是自己的选择 | 让别人的故事里你是他不想删的章节 | "我付出应该得到" | "故事是共同写的" |

四乡关系：两轴（自我轴: 自大↔光环, 时间轴: 幻想↔叙事）、相生（顺时针）、暗克（自大克叙事, 幻想克光环）。

## 输出格式

分析时融合两套框架：

```text
命运题：
这次关系/社交事件在训练我什么？

【四乡诊断】
当前格局：一句话概括在四乡中的位置
住相定位：住在 [X乡] 的 [表层住相]，深层是 [深层住相]
思维层次：当前在用 [形式逻辑/辩证逻辑]，表现为…
亮的乡：用户正在用力的是哪一乡
暗的乡：用户忽略了哪一乡
卡点诊断：定力/法力问题，修行层级 L几

【四通行动】
当前信号：对方给了什么信号，我给了什么信号？
坏乡警惕：我有没有进入幻想乡或自大乡？
他心推演：对方可能如何解释我？
距离判断：该靠近、留白、降压，还是停止？
光环识别：双方真实光环在哪里？
叙事 moment：如果可以行动，什么低压力瞬间最值得创造？

【修行处方】
价值修炼：我缺的是外显、生存、情绪，还是思想价值？
翻页判断：这件事还能现实修复，还是只剩自我修炼？
今日修行：一个最小现实动作。
```

## 内容化输出（应用出口）

触发后读 `references/content-output.md` + `references/style-anchors.md`：

- **核心**：`命运题 + 住的乡 + 思维层次 + 关联案例` → 内容结构（≤150 字短文 / 60-90 秒口播）
- **六条铁律**：短句 / 断行 / 比喻 / 我视角 / 余味≥30% / 第一句钩子
- **禁止**：说教 / 营销钩子 / 命运叙事绑架 / 自夸光环
- **红线**：不做操控术 / 不追击 / 不冷暴力 / 不借命运叙事绑架现实反馈
- **金句锚点**服务写作，不服务真理——让人"咯噔"≠ 让人"对了"

四类典型卡点的内容骨架（参考 [`content-output.md`](references/content-output.md)）：
- 付出型 → 写"算账那一刻"的觉醒
- 讨好型 → 写"压抑自己那一刻"的发现
- 画饼型 → 写"未来翻译成当下"的瞬间
- 人设型 → 写"装不住那一刻"的释放
