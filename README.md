# Destiny Skill

命运 Skill 是一套面向社交直觉、关系判断和自我修炼的 Codex skill。

它不是恋爱技巧，也不是操控术。它把任何关系事件转化成通用训练协议：看见命运给了什么题，警惕自己的幻想和自大，站到别人系统里看自己，调节靠近与远离的距离，并创造能被记住的低压力美好 moment。

## 核心心法

```text
不是抓住对的人，而是修出能识别、靠近、承接和放过对的人的能力。
```

## 适合的问题

- 我没有社交直觉，总是把窗口聊死
- 我遇不到对的人，或者遇到了也抓不住
- 我容易在幻想、自大、证明、追击里失控
- 我想把一次关系事件转成可复盘、可训练、可迁移的能力
- 我想创造美好 moment，但不想滑向操控或讨好

## 四通

```text
观命通：把关系事件看成命运题，而不是自我价值审判。
警惕通：警惕幻想乡和自大乡，不让想象夺权。
他心通：站到对方预测系统里，看自己变成了什么。
距离通：调节注意力、关系权限和现实距离。
```

## 四乡

```text
幻想乡、自大乡：坏乡，要警惕。
光环乡、叙事乡：好乡，要清醒使用。
```

## 使用方式

把本仓库作为 Codex skill 使用时，让 Codex 读取根目录的 `SKILL.md`。

典型触发：

```text
用命运 Skill 分析这次关系事件
我社交直觉弱，总是遇不到对的人
我遇到了但抓不住，帮我看我缺哪一通
我是不是进了幻想乡或自大乡
帮我设计一个低压力、有余味的 moment
```

## 安全边界

命运 Skill 明确不提供以下建议：

- 用焦虑、嫉妒、亏欠、竞争感操控别人
- 把他心通用成读心和绕边界
- 把光环叙事用成包装骗术
- 把距离通用成冷暴力
- 把“命运”“缘分”“修炼”当成追击或越界理由

更好的目标是：

```text
遇不到，就修场域。
遇到了，就修轻重。
抓不住，就修边界。
错过了，就修命。
```
