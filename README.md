# Destiny Skill × 四乡修行法门

命运 Skill 与四乡修行法门是一套面向社交直觉、关系判断和自我修炼的 Agent Skill。

它不是恋爱技巧，也不是操控术。它把任何关系事件转化成通用训练协议：看见命运给了什么题，警惕自己的幻想和自大，站到别人系统里看自己，调节靠近与远离的距离，并创造能被记住的低压力美好 moment。

## 核心心法

```text
不是抓住对的人，而是修出能识别、靠近、承接和放过对的人的能力。
不被四乡所转，是定力。能转他人四乡，是法力。
```

## 两套框架

```text
命运 Skill（四通）：从命运事件切入，训练社交直觉和关系能力——行动框架。
四乡修行法门：从人性结构切入，诊断修行卡点和力量流向——诊断框架。
```

两者合用：先用四乡看清格局，再用四通给出行动。

## 四通

```text
观命通：把关系事件看成命运题，而不是自我价值审判。
警惕通：警惕幻想乡和自大乡，不让想象夺权。
他心通：站到对方预测系统里，看自己变成了什么。
距离通：调节注意力、关系权限和现实距离。
```

## 四乡

```text
自大乡：我怎么看自己。定力 = 放下我执；法力 = 让别人被看见。
幻想乡：我怎么想未来。定力 = 不逃避眼前；法力 = 让人看见未来感。
光环乡：别人怎么看我。定力 = 不被评价牵动；法力 = 给别人舞台。
叙事乡：我为什么在这里。定力 = 不被付出绑架；法力 = 成为不想删的章节。
```

## 底层引擎

```text
住相：把暂时的真相当成永恒的真理，住在里面不走。
住相闭环：住相 → 排异 → 抬杠 → 情绪 → 偏见 → 更深的住相。
思维层次：形式逻辑（住相的温床）vs 辩证逻辑（天然破相）。
```

## 适合的问题

- 我没有社交直觉，总是把窗口聊死
- 我遇不到对的人，或者遇到了也抓不住
- 我容易在幻想、自大、证明、追击里失控
- 我想把一次关系事件转成可复盘、可训练、可迁移的能力
- 我想诊断自己在四乡中的卡点和住相
- 我想从形式逻辑升级到辩证逻辑的修行层次

## 安全边界

命运 Skill 与四乡修行法门明确不提供以下建议：

- 用焦虑、嫉妒、亏欠、竞争感操控别人
- 把他心通用成读心和绕边界
- 把光环叙事用成包装骗术
- 把距离通用成冷暴力
- 把"命运""缘分""修炼"当成追击或越界理由

更好的目标是：

```text
遇不到，就修场域。
遇到了，就修轻重。
抓不住，就修边界。
错过了，就修命。

她不是答案。
她是题。
痛也不是答案。
痛是提醒你：这里有一门功课还没修完。
```
