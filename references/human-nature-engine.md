# 住相人性引擎：四乡的底层操作系统

> 来源：演化方法论 2.0（小说项目），提炼为四乡修行法门的底层支撑。

---

## 核心命题：四乡的每一乡，本质上都是一种住相

住相 = 把暂时的真相当成永恒的真理，然后住进去不走了。

四乡不是四种"关系技巧"，是四种人最容易住进去的相。修行不是学新技巧，是看破自己在住什么相。

---

## 一、住相闭环——为什么人会被困在某一乡

住相一旦形成，会启动自我强化的闭环：

```
住相 → 排异（否定矛盾信息）→ 抬杠（寻找支持证据）→ 情绪（住相被威胁时的防御反应）→ 偏见（观点分歧升级为人格否定）→ 更深的住相
```

**对应到四乡：**

| 乡 | 住什么相 | 排异什么 | 抬杠什么 | 情绪触发点 |
|---|---|---|---|---|
| 自大乡 | "我的判断是对的" | 别人和自己不同的视角 | 支持自己观点的证据 | 被否定、被忽视 |
| 幻想乡 | "未来会更好的" | 眼前必须面对的现实 | 选择性关注积极信号 | 被提醒"你还没做" |
| 光环乡 | "我在别人眼里是X" | 与自我形象矛盾的信息 | 强化人设的反馈 | 人设被拆穿、被看低 |
| 叙事乡 | "我付出了这么多" | 对方不按自己剧本走的反应 | "我都这样了他还不…" | 付出被无视、被辜负 |

**诊断价值：** 当用户描述一种反复出现的情绪反应时，沿闭环回溯——情绪背后是什么偏见？偏见背后是什么抬杠？抬杠背后是什么排异？排异背后是什么住相？——就能定位他住在哪一乡。

---

## 二、住相三维九宫格——精确定位你在住什么

住相按三个维度（自我/他人/世界）和三种核心需求（价值感/控制感/归属感）交叉，形成九宫格。

**与四乡的映射关系：**

| 维度\需求 | 价值感 | 控制感 | 归属感 |
|---|---|---|---|
| **关于自我** | 我必须证明自己有价值 | 我必须掌控一切才安全 | 我必须融入才完整 |
| **关于他人** | 别人的认可定义我的价值 | 别人都想控制我 | 别人终将背叛我 |
| **关于世界** | 世界会奖励有价值的人 | 世界是可被理解的 | 世界是善意的 |

**四乡对应的住相区间：**

- **自大乡** → 主要落在"关于自我"维度：价值感（我要证明自己对）和控制感（我要掌控对话）
- **光环乡** → 主要落在"关于他人"维度：价值感（别人的评价定义我）和控制感（我要管理别人怎么看我）
- **幻想乡** → 主要落在"关于世界"维度：控制感（世界应该按我想象的发展）和归属感（世界应该有我的位置）
- **叙事乡** → 跨越"自我+他人"：归属感（只要我付出别人就会爱我）和价值感（我的付出应该有回报）

**诊断价值：** 用户说"我觉得他不尊重我"——这是"关于他人 × 价值感"的住相，对应光环乡。用户说"我都做了这么多他还这样"——这是"叙事乡 × 归属感"的住相。精确定位住相的维度和需求，就能找到修行的切口。

---

## 三、表层住相与深层住相——看见表面下的冰山

每个住相都有两层：

- **表层住相**：自己意识到的、可以说出口的信念。"我希望他对我好。"
- **深层住相**：不自知的、驱动行为的真正原因。"我害怕自己不值得被好好对待。"

**四乡的表层与深层：**

| 乡 | 常见表层住相 | 常见深层住相 |
|---|---|---|
| 自大乡 | "我的判断是对的" | "我害怕承认自己错了就意味着我不够好" |
| 幻想乡 | "以后会更好的" | "我承受不了现在，所以只能逃进未来" |
| 光环乡 | "我应该被尊重" | "如果别人不尊重我，我就不知道自己是谁" |
| 叙事乡 | "我付出了这么多" | "如果我的付出没有意义，那我做的一切都是白费" |

**修行价值：** 四乡法门的两层功夫（定力/法力），如果只在表层修，是治标。修到深层住相，才是治本。比如：自大乡的定力不是"学会闭嘴"（表层），是"接受自己可以不总是对的"（深层）。

## 四、执念是中性的燃料——不评判，只看流向

**校正一个关键认知：住相不是坏的。**

执念是人类一切行为的燃料。没有执念的人不会做任何事。关键在于它驱动你走向哪里：

| 执念的流向 | 在四乡中的表现 |
|---|---|
| 驱动行动 | 自大乡的执念让你去证明自己 → 做出了成绩 |
| 扭曲感知 | 自大乡的执念让你只看到支持自己的证据 → 脱离现实 |
| 碰撞产生冲突 | 你的自大乡和对方的叙事乡碰撞 → 双方都觉得委屈 |
| 被看破 | 意识到"我不是对的，我只是住在了'对'里面" → 定力真正生起 |
| 被成全 | 你的执念和现实对齐 → 暂时满足，但很快会形成新执念 |
| 螺旋上升 | 看破旧执念 → 形成新执念 → 更接近真实 |

**修行价值：** 四乡修行不是消灭执念，是觉察执念的流向。当你发现自己住在某一乡时，不要急着"破"，先问：这个执念在驱动我做什么？它扭曲了什么？它在保护我什么？

## 五、住相碰撞升级路径——关系是怎么一步步变糟的

两个人的住相碰撞，会沿着固定路径升级：

```
初遇（表面和谐，住相未触发）
  ↓
事件触发（某件事激活了双方的住相）
  ↓
分歧出现（各自按住相解读事件，得出不同结论）
  ↓
试图说服（抬杠阶段）
  ↓
情绪升级（住相受到威胁，启动防御）
  ↓
关系破裂 / 一方住相松动 / 找到更高维度的和解
```

**四乡碰撞的典型路径：**

- **自大乡 × 叙事乡**：你太关注自己的判断（自大），对方觉得自己的付出不被看见（叙事），分歧升级为"你从来不听我说话" vs "你说的都是废话"。
- **幻想乡 × 光环乡**：你画了一个未来的饼（幻想），对方开始信了，但你的真实状态和人设不符（光环），对方发现真相后光环碎裂，幻想也跟着碎。

**诊断价值：** 用户描述一段关系冲突时，先判断双方各自住在哪个乡的什么相上，然后定位他们在碰撞路径的哪个阶段。不同阶段的修法不同：
- 分歧阶段 → 修的是"看见对方的住相有他的道理"（自大乡定力）
- 抬杠阶段 → 修的是"分清我在说服还是在理解"（自大乡定力 + 叙事乡定力）
- 情绪阶段 → 先止血，不在这个阶段讲道理（等住相冷却再处理）
- 破裂边缘 → 至少一方的住相需要松动，这通常需要一个强力的"发现"事件

## 六、四层感知与修行深度

修行的深度对应四个层级：

| 层级 | 修的是什么 | 在四乡中的表现 |
|---|---|---|
| L1 行为层 | 改变外在行为 | 学会闭嘴、学会夸奖、学会不反驳 |
| L2 情感层 | 处理情绪反应 | 觉察到住相被触发时的愤怒/委屈/恐惧，不被它带走 |
| L3 认知层 | 看破住相本身 | "我不是对的，我只是住在了'对'里面" |
| L4 存在层 | 触及空性 | 连"我要修行"这个住相也看到了，一笑而过 |

**对应到定力/法力：**

- L1-L2 修的是**定力**的基础：不被住相的外在表现牵动
- L3 修的是**定力**的核心：看到住相本身，而不只是住相的产物
- L4 是**法力**的真正来源：空性不是什么都没有，是什么都能是——你不再被任何一个乡的框架限制，才能在四个乡之间自如切换

## 七、空性与妙用——四乡的最高境界

空性：每一乡都没有固定的本质。自大不是坏的，幻想不是逃避，光环不是虚伪，叙事不是绑架。它们的意义完全取决于你如何使用它们。

妙用：正因为每一乡本身是空的，它才什么都能是。

- 自大乡的空性：放下"我必须对"之后，你的判断力依然在，只是不再被"我"这个滤镜扭曲
- 幻想乡的空性：停止逃避之后，你的想象力依然在，只是从止痛药变成了导航仪
- 光环乡的空性：拆掉人设之后，你的魅力依然在，只是不再需要维护
- 叙事乡的空性：放下"我应该被回报"之后，你的付出依然在，只是从交易变成了选择

**这就是四乡修行的终极指向：不是修到没有住相，是修到能在住相和空性之间自由出入。**

用，就住进去——为了那个当下全力以赴。
不用，就出来——不带走任何东西。

## 八、诊断增强：用住相理论升级四乡诊断

在原有四乡诊断流程的基础上，增加住相维度：

### 诊断步骤增强

1. **定乡**（原有）：用户在哪个乡用力最多/最暗？
2. **定相**（新增）：他具体住在什么相上？用三维九宫格精确定位。
3. **定层**（新增）：表层住相是什么？深层住相是什么？
4. **定环**（新增）：他的住相闭环走到哪一步了？（排异？抬杠？情绪？偏见？）
5. **定碰撞**（新增）：如果涉及两人，双方住相碰撞在升级路径的哪个阶段？
6. **定流向**（新增）：这个执念在驱动他做什么？扭曲了什么？保护了什么？
7. **定修行层级**（新增）：用户当前在L几？需要推到L几？

### 诊断输出增强

```
【当前格局】一句话概括
【住相定位】住在 [X乡] 的 [表层住相]，深层是 [深层住相]，属于九宫格的 [维度×需求]
【闭环状态】住相闭环走到了 [排异/抬杠/情绪/偏见] 阶段
【碰撞分析】（如涉及两人）双方住相在升级路径的 [阶段]
【执念流向】这个执念在驱动 [行为]，扭曲了 [感知]，保护着 [深层需求]
【修行层级】当前在 L[X]，建议推到 L[Y]
【修行建议】（基于以上全部分析给出）
```

## 十、形式逻辑 vs 辩证逻辑：住相的操作系统

住相是形式逻辑的产物。辩证逻辑天然破相。

**形式逻辑**：单变量、线性、凭感觉下判断。把世界切成固定范畴（对/错、好/坏、我/他），住在其中一个范畴里不走。大部分人在这一层。

**辩证逻辑**：多变量、多维度、能同时持有矛盾。允许同一事物在不同框架下有不同真相。不执着于任何单一范畴。少部分高手在这一层。

| 乡 | 形式逻辑（住在这里） | 辩证逻辑（看破住相） |
|---|---|---|
| 自大乡 | "我对，你错" | "我可以同时对和错" |
| 幻想乡 | "以后会更好" | "不确定，但我依然行动" |
| 光环乡 | "我是X" | "我是流动的、多面的" |
| 叙事乡 | "我付出应该得到" | "故事是共同写的" |

**关键洞察：** 形式逻辑的定力 = "我忍住了"（但还是痛）。辩证逻辑的定力 = "我根本不需要忍"（因为同时看到了多个面）。大部分人的修行停在形式层。

**诊断时必问：** 用户当前在用形式逻辑还是辩证逻辑处理这个问题？如果卡在形式层，修行方向就是从形式走向辩证。具体的转折提示见 [full-framework.md](full-framework.md) 中每乡的"形式版 vs 辩证版"章节。
